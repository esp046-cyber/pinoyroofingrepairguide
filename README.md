# pinoyroofingrepairguide
Features:
5 Main Sections: Uri ng Bubong (Roof Types), Mga Problema (Problems), Pagkukumpuni (Repairs), Materyales (Materials), and Mga Tip (Tips)
Roofing theme: Brown/rust colors mimicking roof materials with gradient backgrounds
Small font titles with 1% margins as requested
Mobile-optimized: Tab navigation, touch-friendly buttons, responsive design
No search bar or 3D elements as specified
Filipino language throughout for local users
Content covers:
Common Philippine roof types (Yero, Long Span, Asbesto, Nipa)
Common problems and solutions
Step-by-step repair guides with estimated costs in pesos
Materials list with local prices
Safety tips and maintenance advice
Contractor selection tips
The app is ready to use on mobile devices with smooth tab switching and a clean, professional layout perfect for quick reference during roofing projects!
